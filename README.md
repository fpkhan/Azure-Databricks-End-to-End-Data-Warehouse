# Azure-Databricks-End-to-End-Data-Warehouse
An end-to-end data engineering pipeline built on Microsoft Azure and Apache Spark (Databricks) implementing a Medallion Architecture. Features incremental data ingestion via Autoloader, enterprise data governance via Unity Catalog, PySpark windowing functions, and automated SCD Type 1 &amp; Type 2 (DLT) data warehousing schemas.
